View触摸事件的分发

Android手机最大的器官就是触摸屏了，所以掌握好触摸操作是非常重要的

在分析触摸事件的分发之前，我们先看一下什么是触摸事件：

MotionEvent

在手指接触屏幕后所产生的一系列事件中，典型的有下面几种

  1. ACTION_DOWN –手指刚接触屏幕
  2. ACTION_MOVE–手指在屏幕上滑动
  3. ACTION_UO–手指从屏幕上松开的一瞬间

View的位置信息

获取点击位置的坐标：MotionEvent的方法
得到相对于父view的坐标

  1. getX()
  2. getY()

得到相对于屏幕的坐标：

  1. getRawX()
  2. getRawY()

获取控件的坐标：

  • getLeft() 获取左上角的横坐标
  • getTop()获取左上角纵坐标
  • getRight()获取右下角横坐标
  • getBttom()获取右下角纵坐标

每个View都是矩形

点击事件的分发过程由三个非常重要的方法完成：

  1. public boolean dispatchTouchEvent(MotionEvent ev)
  2. public boolean onInterceptTouchEvent(MotionEvent ev)
  3. public boolean onTouchEvent(MotionEvent event)

三个方法的分布情况

在View中包含以下两个方法：

public boolean dispatchTouchEvent(MotionEvent ev)；

public boolean onTouchEvent(MotionEvent ev);

在ViewGroup中包含以下三个方法：

public boolean dispatchTouchEvent(MotionEvent ev)；

public boolean onInterceptTouchEvent(MotionEvent ev);

public boolean onTouchEvent(MotionEvent ev);

在Activity中包含以下两个方法：

public boolean dispatchTouchEvent(MotionEvent ev)；

public boolean onTouchEvent(MotionEvent ev);
